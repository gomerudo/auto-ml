Contribute¶

We encourage the Open Source community to contribute to this project. As of today, the project is in an alpha stage, since the development ran for 2 months only. However, it is stable enough to run for general requirements.

If you want to contribute to make this project more robust either in the software engineering area or in its data science components, please contact any of the persons listed below:
